## Project Overview

Jarvis is designed to act as a smart virtual co-pilot capable of perceiving, reasoning, and responding dynamically.  
It uses LiveKit for real-time audio and video streaming and OpenAI APIs for natural conversation, decision-making, and multimodal understanding.

---

## Key Features
- Real-time voice recognition and response generation  
- Vision support for image and live input analysis  
- Agentic reasoning for task automation and adaptive responses  
- Modular plugin system for easy feature expansion  
- Context-aware conversations that feel natural and continuous
